xipy.io.vtk_polydata_reader.load_vtk_polydata(self, fname)¶
Read a vtk polydata file. Mayavi can do this as well but it generates some kind of weird error. Because the vtk format is very easy to implement we have implemented here a very simple polydata reader. Polydata are usually used for surface representations.
